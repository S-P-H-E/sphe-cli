# The Sphe CLI
This command line tool is used to quickly generated Typescript projects.

To use the cli, run:
```
npx sphe-cli
```

To clone and run the cli, run:
```
git clone https://github.com/S-P-H-E/sphe-cli.git
npm install

npm run dev
```

The current supported project templates are:
- Expo
- Node
- Nuxt
